# Images Directory

Place your profile image here:

## Recommended:
- **File name**: `profile.jpg` or `profile.webp` (WebP preferred for performance)
- **Size**: 400x400px or higher (square format works best)
- **Format**: WebP, JPG, or PNG
- **Quality**: High resolution for professional look

## Location:
```
public/images/
├── profile.jpg    ← Your professional photo here
└── README.md      ← This file
```

## Tips for best results:
- Use a professional headshot with good lighting
- Plain or simple background works best
- Face should be clearly visible
- Smile and look confident
- High resolution (at least 400x400px)
